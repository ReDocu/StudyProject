콘텐츠 일괄 추가

과목은 고정 목록에서 선택하고, 주제/소주제 콘텐츠를 여러 개 추가한 뒤 각 콘텐츠 내부에서 키워드 블록을 원하는 만큼 늘려 한 번에 저장하세요.

입력 방식

각 콘텐츠 블록은 주제 / 소주제 / 본문 / 키워드 목록으로 구성됩니다. 입력한 줄바꿈과 띄어쓰기 흐름은 실제 학습 페이지 출력에서도 최대한 비슷하게 유지됩니다. 키워드 목록은 [키워드 박스 + 키워드 설명 박스] 세트로 관리되며, 여러 블록을 추가해 일괄 저장할 수 있습니다.
