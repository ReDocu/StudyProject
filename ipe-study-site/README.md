# 정보처리기사 학습 웹사이트

완전 로컬 기반(HTML/CSS/Vanilla JS)으로 동작하는 정보처리기사 학습 웹앱입니다.

## 실행 방법
1. 압축 해제
2. `index.html` 실행
3. 데이터는 브라우저 LocalStorage에 저장

## 이번 업데이트 반영 사항
- 필기/실기 학습 페이지에 `학습 완료 후 다음으로` 기능 추가
- 필기/실기 고정 과목 구조 적용
- 관리자 페이지에서 필기/실기 콘텐츠를 여러 블록으로 일괄 저장 가능
- 관리자 페이지의 과목 입력을 자유 입력에서 고정 선택 방식으로 변경
- JSON import 시에도 고정 과목 구조를 자동 보정

## 고정 과목 구조
### 필기
- 소프트웨어 설계
- 소프트웨어 개발
- 데이터베이스 구축
- 프로그래밍 언어 활용
- 정보 시스템 구축 관리

### 실기
- 요구사항 확인
- 데이터 입출력 구현
- 통합 구현
- 서버 프로그램 구현
- 인터페이스 구현
- 화면 설계
- 애플리케이션 테스트 관리
- SQL 응용
- 소프트웨어 개발 보안 구축
- 프로그래밍 언어 활용
- 응용 SW 기초 기술 활용
- 제품 소프트웨어 패키징
